# Milestone

Milestone is a mobile-first long-term goal pomodoro prototype.

It turns a long-term goal into a staged roadmap, expands only the current phase into daily tasks, supports task check-ins, pomodoro focus sessions, plan revision, deadline changes, and history archives.

## Run

Open `index.html` in a browser.

No build step or dependencies are required. App data is stored in browser `localStorage`.

## Files

- `index.html` - app shell and dialogs
- `styles.css` - mobile UI styles
- `app.js` - planning, check-in, timer, archive, and storage logic
